Welcome to BCGov Rocket.Chat!


User How-Tos:

  • Complete your business information if missing, and upload an avatar.
  • Team members cannot join? Invite new GitHub users or read the user guide.
  • iOS issue with too many redirects? Clear cache on safari helps. See instruction.
  • Seeing Internal Service Error from KeyCloak? Have you tried to clear browser cache and re-login? Still have the problem? Please check if your IDIR account has an email associated with it. If not, either request to update your IDIR account or use an GitHub account to login to RocketChat.

Channel How-Tos:

  • To use the default channels, please read each channel's description.
  • DevOps platform-related channels will be created and moderated by the DevOps Team Services team. These channels will be prefixed with devops-.
  • To create team-specific channels:
    • Please read through the chat conventions.
    • Team channel names should be prefixed with <Team Name>-. Please be sure to provide a channel description.
    • Please note that channels with confusing or inappropriate names may be renamed by the platform admins.

Native desktop and mobile apps are available. See below:

  • Rocket.Chat desktops apps are available to download here.
  • Native mobile apps are available from Google Play and App Store.
  • To sign in to Rocket.Chat on apps, please connect to server chat.developer.gov.bc.ca
